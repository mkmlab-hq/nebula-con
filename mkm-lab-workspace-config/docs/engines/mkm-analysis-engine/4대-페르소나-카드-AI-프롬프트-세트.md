- [ ] 중앙 요소의 명확한 표현

- [ ] 사용자 얼굴 통합 가능한 구조
- [ ] NFT 발행 적합성

- **반복 생성**: 각 프롬프트로 3-5회 생성 후 최고 품질 선택
