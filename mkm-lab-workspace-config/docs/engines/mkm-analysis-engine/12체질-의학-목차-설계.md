### **핵심 철학**
```
🏗️ 새로운 연구 패러다임
├── GitHub: "설계도" 보관소 (코드 + 색인 데이터)
├── GCS: "원자재" 대형 창고 (대용량 연구 데이터)
└── 자동화: GitHub Actions를 통한 연구 프로세스 자동화
```

      runs-on: ubuntu-latest
      steps:
        - name: Collect research data
