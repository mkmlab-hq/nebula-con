### 🏗️ 파이프라인 아키텍처

- **벡터화**: TF-IDF + 해시 기반 임베딩
- **저장**: JSON + NumPy + 메타데이터
- **검색**: 코사인 유사도 기반
- **RAG**: 기본 프롬프트 + 컨텍스트 결합
